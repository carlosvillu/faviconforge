# Términos de Servicio

**Última actualización: Enero 2025**

## 1. Aceptación de los Términos

Al acceder y usar FaviconForge (el "Servicio"), aceptas estar sujeto a estos Términos de Servicio ("Términos"). Si no estás de acuerdo con estos Términos, por favor no uses el Servicio.

## 2. Descripción del Servicio

FaviconForge es una aplicación web que permite a los usuarios crear, personalizar y descargar favicons para sus sitios web. El Servicio ofrece niveles gratuitos y premium con diferentes características y capacidades.

## 3. Niveles Gratuito y Premium

### Nivel Gratuito
- Acceso a características básicas de generación de favicons
- Formatos de icono estándar (ICO, PNG)
- Opciones de personalización limitadas
- Sin restricciones de uso comercial

### Nivel Premium
- Opciones avanzadas de personalización incluyendo manifests personalizados
- Soporte prioritario
- Licencia de uso comercial
- Formatos y tamaños de archivo adicionales
- Acceso a plantillas y características premium
- Acceso permanente con un pago único (sin cargos recurrentes)

El pago del nivel Premium se procesa de forma segura a través de Stripe como un pago único que otorga acceso permanente a todas las características premium mientras el servicio esté operativo. No hay garantía de que el servicio estará disponible indefinidamente.
